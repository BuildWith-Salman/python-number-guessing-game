# python-number-guessing-game
A simple Python command-line Number Guessing Game where the player has five chances to guess a randomly generated number between 1 and 100.


# Python Number Guessing Game

A simple command-line game built using Python.

## Features

- Random number generation
- 4 guessing attempts
- Hint after every wrong guess
- Displays win or lose message
- Beginner-friendly Python project

## Technologies Used

- Python 3
- random module

## How to Run

Clone the repository

git clone https://github.com/yourusername/python-number-guessing-game.git

Run

python guess.py
